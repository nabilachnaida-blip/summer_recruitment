## 🎨 Customization Examples

### Update Google Form URL
1. Open `index.html` in text editor
2. Find: `https://forms.google.com/placeholder`
3. Replace with your actual Google Form URL
4. Save file and refresh browser

### Change Company Colors
1. Open `style.css` in text editor
2. Modify the CSS variables at the top:
```css
--primary-blue: #0D6FA3;  /* Change this color */
--primary-teal: #00B3B8;  /* Change this color */
```
3. Save and refresh browser

### Edit Text Content
1. Open `index.html` in text editor
2. Find the section you want to edit
3. Update the text
4. Save and refresh
